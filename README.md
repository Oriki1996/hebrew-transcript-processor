# מעבד תמלילים אקדמי

כלי ווב חד-קובץ ללא שרת, המעבד תמלילים גולמיים בעברית למסמכים אקדמיים מובנים ומעוצבים — באמצעות ה-API של Anthropic.

---

## תיאור

מעבד התמלילים האקדמי הוא יישום ווב עצמאי הפועל ישירות בדפדפן, ללא צורך בהתקנה, שרת, או תשתית ענן. הוא מקבל תמליל גולמי (מקובץ PDF, Word, TXT, או הדבקה ידנית), מזהה את סוג התוכן, ומפיק ממנו מסמך מעובד ואיכותי בעברית אקדמית תקנית.

הכלי תומך בארבעה סוגי תמלול: **הרצאה אקדמית**, **ישיבת עבודה**, **ראיון**, ו**פאנל**. לכל סוג מוגדרות הנחיות עריכה ייעודיות המותאמות לאופי החומר ולמטרת המסמך.

---

## תכונות עיקריות

### חילוץ טקסט מקבצים
- **PDF** — מנוע חילוץ מלא הכתוב ב-JavaScript טהור, ללא ספריות חיצוניות:
  - דחיסת FlateDecode (zlib ו-raw deflate) באמצעות `DecompressionStream` הנייטיב של הדפדפן
  - ניתוח זרמי תוכן PDF מסוג `BT...ET` עם תמיכה מלאה באופרטורים `Tj`, `TJ`, `'` ו-`"`
  - תמיכה במחרוזות ליטרל `(...)` עם פענוח תווי בריחה מלא (כולל אוקטלי `\ddd`)
  - תמיכה במחרוזות הקס `<...>`
  - זיהוי וקידוד UTF-16BE עם BOM — קריטי לקבצי PDF בעברית מודרניים
  - מנגנון fallback לקבצי PDF פשוטים ולא דחוסים
- **DOCX** — חילוץ ישיר מקובץ ZIP בפורמט Office Open XML, ללא ספריות חיצוניות
- **TXT** — קריאה ישירה עם `File.text()`

### עיבוד חכם לקבצים ארוכים
הכלי מחשב אוטומטית את מספר החלקים הנדרש לפי אורך התמליל, ומחלק את הטקסט לקטעים מוגבלי אורך לפני השליחה ל-API. כך נמנעות שגיאות token limit גם בתמלילים ארוכים במיוחד.

### אחסון מפתח API מקומי
מפתח ה-API נשמר ב-`localStorage` של הדפדפן. הוא נטען אוטומטית בפתיחת הכלי ומעודכן בכל שינוי. אין שמירה בשרת כלשהו.

### עיצוב אקדמי קפדני
- הפקת HTML מובנה עם כותרות היררכיות (`h1` / `h2` / `h3`)
- פסקאות פרוזה רציפות, הדגשת מושגי מפתח
- פרק "מושגים מרכזיים" בסוף כל הרצאה
- ייחוס מדויק לחוקרים ותיאוריות
- טיפול בשפה מעורבת (עברית–אנגלית) לפי כללים אחידים
- ניקוי אוטומטי של שפה מדוברת, חזרות, ומילות מילוי

## שימוש

1. **פתח את הקובץ** — גרור את `hebrew_transcript_processor_with_pako.html` לדפדפן, או לחץ עליו פעמיים.

2. **הכנס מפתח API** — בשורת הכותרת העליונה, הדבק את מפתח ה-Anthropic שלך (מתחיל ב-`sk-ant-`). המפתח יישמר אוטומטית לשימוש הבא.

3. **טען תמליל** — בחר אחת מהאפשרויות:
   - הדבק טקסט גולמי ישירות לתיבת הטקסט
   - לחץ **"העלה TXT"** לטעינת קובץ טקסט
   - לחץ **"העלה Word/PDF"** לטעינת קובץ DOCX או PDF

4. **זהה ועבד** — לחץ **"זהה סוג ועבד ←"**. הכלי ינתח את סוג התמליל ויציג את המלצותיו.

5. **אשר והתחל** — בדוק את הזיהוי (ניתן לשנות ידנית), ולחץ **"אשר והתחל עיבוד ←"**. סרגל ההתקדמות יעדכן בזמן אמת.

6. **ייצא את התוצאה** — בסיום העיבוד ניתן:
   - להעתיק את הטקסט המלא ללוח
   - לייצא לקובץ TXT
   - לפתוח עיבוד חדש

---

## אבטחה ופרטיות

> **מפתח ה-API והתמלילים שלך אינם עוברים דרך שום שרת ביניים.**

- כל העיבוד מתבצע **מקומית בדפדפן בלבד**
- בקשות ה-API נשלחות **ישירות** מהדפדפן שלך לשרתי Anthropic — ללא proxy, ללא שרת, ללא logging חיצוני
- מפתח ה-API נשמר אך ורק ב-`localStorage` של הדפדפן שלך — לא נשלח לאף גורם שלישי
- הקובץ יכול לפעול במצב offline לחלוטין (למעט קריאות ה-API עצמן)
